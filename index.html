C-Shell

C-Shell is a command-line interprer that mimics a Linux/Unix command line made using C. This shell supports listing files and directories in the file system, history of executed commands, and environment variables. Upon calling an unsupported function, CShell will create a fork to a child process, where the command will be passed and executed by the normal Linux/Unix terminal.
